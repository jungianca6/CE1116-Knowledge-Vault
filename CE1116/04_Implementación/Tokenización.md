# Tokenización


---

## 📚 Idea/Concepto 

La tokenización es el proceso de dividir un texto en unidades más pequeñas llamadas tokens. Estos tokens pueden ser palabras, partes de palabras o caracteres, y son la forma en que un modelo de lenguaje puede procesar texto como datos numéricos.
