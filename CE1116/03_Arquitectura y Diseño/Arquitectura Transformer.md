# Arquitectura Transformer


---

## 📚 Idea/Concepto 

La arquitectura Transformer es un tipo de modelo de redes neuronales diseñado para procesar secuencias de datos, especialmente texto. Utiliza mecanismos de atención para analizar la relación entre palabras dentro de un contexto y es la base de muchos modelos de lenguaje modernos.
