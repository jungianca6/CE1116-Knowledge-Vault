# Función de Activación


---

## 📚 Idea/Concepto 

Una función de activación es una operación matemática aplicada a la salida de una neurona en una red neuronal para determinar si la información debe transmitirse a la siguiente capa. Permite que la red neuronal pueda aprender patrones complejos.
