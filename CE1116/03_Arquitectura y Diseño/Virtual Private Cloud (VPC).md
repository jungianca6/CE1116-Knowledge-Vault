## 📚 Idea/Concepto 

Una Virtual Private Cloud (VPC) es una red virtual lógicamente aislada dentro de una nube pública, construida sobre infraestructura compartida mediante redes definidas por software (SDN). No equivale a una nube privada (que usa hardware dedicado). La VPC permite a una empresa controlar su propio espacio de red definiendo subredes, tablas de enrutamiento, gateways (NAT, VPN) y firewalls (grupos de seguridad/ACL) todo ello gestionado por API o CLI. El aislamiento es lógico, no físico. Además, la VPC sirve como base para la nube híbrida, al permitir conectar la red local ("on-premise") mediante VPN o conexiones dedicadas (como AWS Direct Connect). Así, se combina la escalabilidad y conveniencia de la nube pública con un nivel de control similar al de una nube privada, pero sin el hardware exclusivo.
