# Ventana de Contexto


---


## 📚 Idea/Concepto 

La ventana de contexto es la cantidad máxima de tokens que un modelo de lenguaje puede considerar al mismo tiempo al procesar información. Esta limitación determina cuánta información puede tener en cuenta para generar una respuesta.
